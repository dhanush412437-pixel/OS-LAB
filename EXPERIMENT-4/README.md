# Experiment 4 – CPU Scheduling Algorithms

## Programs

- FCFS (`fcfs.c`, `fcfs.sh`)
- SJF (`sjf.c`, `sjf.sh`)
- Priority Scheduling (`priority.c`, `priority.sh`)
- Round Robin (`round_robin.c`, `round_robin.sh`)

## Compile C Programs

```bash
gcc fcfs.c -o fcfs
./fcfs

gcc sjf.c -o sjf
./sjf

gcc priority.c -o priority
./priority

gcc round_robin.c -o round_robin
./round_robin
```

## Run Shell Scripts

```bash
chmod +x *.sh
./fcfs.sh
./sjf.sh
./priority.sh
./round_robin.sh
```
