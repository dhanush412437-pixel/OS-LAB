
# Experiment 3 – System Calls

## Programs

- fork_getpid_wait_exit.c
- fork_getpid_wait_exit.sh
- wait_system_call.c
- wait_system_call.sh
- close_system_call.c
- close_system_call.sh

## Compilation

### C Programs

```bash
gcc fork_getpid_wait_exit.c -o fork
./fork

gcc wait_system_call.c -o wait
./wait

gcc close_system_call.c -o close
./close
```

### Shell Scripts

```bash
chmod +x fork_getpid_wait_exit.sh
./fork_getpid_wait_exit.sh

chmod +x wait_system_call.sh
./wait_system_call.sh

chmod +x close_system_call.sh
./close_system_call.sh
```

## Requirements

- GCC Compiler
- Bash Shell
- Linux / Kali Linux
